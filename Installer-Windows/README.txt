Install guide for Windows:

1. Execute the Python file and follow the instructions.
2. Execute the "Python-Request-Tkinter-installer.bat" file. It contains content for running our Script.

Now open the folder that our python files are inside with the Terminal. You can locate to the Folder or right click the empty places from your folder 
and click "Open in Terminal". After that type "python3 start.py".
